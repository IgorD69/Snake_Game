using GameImage = std::vector<std::string>

Tip alias pentru reprezentarea unei imagini de joc.

GameImage este un vector de șiruri de caractere, unde fiecare șir reprezintă o linie din imagine. Fiecare caracter din șir reprezintă un pixel/celulă din joc.

Exemple de caractere:

  • '.' = spațiu gol
  • '#' = perete/margine
  • 'O' = segment șarpe
  • '@' = măr
GameImage image(20, std::string(40, '.')); // Imagine 40x20 goală
image[10][20] = '@'; // Plasează măr la poziția (20, 10)
