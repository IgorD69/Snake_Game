Detailed Description

Clasă pentru reprezentarea hărții jocului Snake.

Map stochează dimensiunile hărții de joc și oferă funcționalități pentru verificarea limitelor și compararea hărților.

Note
Coordonatele valide sunt în intervalul [0, width) și [0, height)
Dimensiunea implicită este 40x40 unități
