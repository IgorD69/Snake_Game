Detailed Description

Interfață abstractă pentru sistemele de input.

AbstractListener definește contractul pentru toate clasele care gestionează input-ul de la utilizator. Folosește pattern-ul Strategy pentru a permite schimbarea sistemului de input la runtime.

Implementările concrete pot include:

  • CmdListener pentru input în terminal (folosind termios)
  • WindowListener pentru input în ferestre grafice
Note
Toate metodele sunt pure virtual și trebuie implementate de clasele derivate

Metodă non-blocantă care verifică instant starea unei taste. Util pentru verificări rapide în loop-ul de joc.

}
