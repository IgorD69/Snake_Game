Detailed Description

Clasă pentru reprezentarea șarpelui în joc.

Player gestionează toate aspectele legate de șarpe: poziția, mișcarea, creșterea atunci când mănâncă mere, și verificarea coliziunilor cu sine sau cu pereții.

Șarpele este format dintr-o listă de segmente (Position), unde primul segment este capul șarpelui.

Note
Capul șarpelui este întotdeauna la indexul 0 în vectorul de segmente

Parameters
directionDirecția în care se mișcă șarpele

Mută șarpele cu o unitate în direcția specificată. Capul se mișcă în direcția dată, iar restul segmentelor urmează capul. Ultimul segment dispare (dacă nu s-a mâncat un măr).
