Detailed Description

Interfață abstractă pentru sistemele de randare.

AbstractRenderer definește contractul pentru toate clasele care gestionează afișarea grafică a jocului. Folosește pattern-ul Strategy pentru a permite schimbarea sistemului de randare.

Implementările concrete pot include:

  • CmdRenderer pentru randare în terminal (ncurses)
  • Renderer pentru randare cu Raylib
  • SDLRenderer pentru randare cu SDL
Note
Toate metodele sunt pure virtual și trebuie implementate de clasele derivate

Parameters

bottomRightColțul dreapta-jos al regiunii de desenat (nefolosit în implementarea actuală)
imageImaginea de desenat (matrice de caractere)

Randează imaginea jocului (harta, șarpele, mărul) pe ecran. Fiecare caracter din imagine este interpretat și desenat cu culoarea/stilul corespunzător:

  • '.' = gol (nu se desenează)
  • '#' = perete (alb/gri)
  • 'O' = șarpe (verde)
  • '@' = măr (roșu)
GameImage image(20, std::string(40, '.'));
image[10][20] = '@'; // Măr
image[10][21] = 'O'; // Șarpe
renderer->DrawImage(Position(0, 0), Position(800, 600), image);
