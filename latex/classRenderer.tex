\doxysubsection{Detailed Description}
Implementare a randării grafice folosind Raylib. 

\doxylink{classRenderer}{Renderer} gestionează afișarea grafică a jocului într-\/o fereastră dedicată folosind biblioteca Raylib. Oferă\+:
\begin{DoxyItemize}
\item Fereastră grafică redimensionabilă
\item Randare pe bază de grilă (grid-\/based)
\item Culori diferite pentru elemente (verde=șarpe, roșu=măr)
\item Text pentru scor și mesaje
\item Control al frame rate-\/ului
\end{DoxyItemize}

Sistemul de coordonate\+:
\begin{DoxyItemize}
\item Fiecare celulă din joc ocupă cell\+Size x cell\+Size pixeli
\item Coordonatele (0,0) sunt în colțul stânga-\/sus
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Necesită biblioteca Raylib instalată și linkată 
\end{DoxyNote}
