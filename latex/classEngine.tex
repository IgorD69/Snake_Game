\doxysubsection{Detailed Description}
Clasă principală pentru gestionarea jocului Snake. 

\doxylink{classEngine}{Engine} este nucleul jocului care coordonează toate componentele\+:
\begin{DoxyItemize}
\item Inițializarea hărții, șarpelui și mărului
\item Loop-\/ul principal de joc cu input, update, și render
\item Detectarea coliziunilor cu pereții și cu propriul corp
\item Gestionarea stării de game over și restart
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Folosește biblioteca Raylib pentru randare și input 
\end{DoxyNote}
