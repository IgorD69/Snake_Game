\doxysubsection{Detailed Description}
Interfață abstractă pentru sistemele de input. 

\doxylink{structAbstractListener}{Abstract\+Listener} definește contractul pentru toate clasele care gestionează input-\/ul de la utilizator. Folosește pattern-\/ul Strategy pentru a permite schimbarea sistemului de input la runtime.

Implementările concrete pot include\+:
\begin{DoxyItemize}
\item \doxylink{classCmdListener}{Cmd\+Listener} pentru input în terminal (folosind termios)
\item Window\+Listener pentru input în ferestre grafice
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Toate metodele sunt pure virtual și trebuie implementate de clasele derivate 
\end{DoxyNote}
