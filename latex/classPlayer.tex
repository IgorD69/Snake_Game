\doxysubsection{Detailed Description}
Clasă pentru reprezentarea șarpelui în joc. 

\doxylink{classPlayer}{Player} gestionează toate aspectele legate de șarpe\+: poziția, mișcarea, creșterea atunci când mănâncă mere, și verificarea coliziunilor cu sine sau cu pereții.

Șarpele este format dintr-\/o listă de segmente (\doxylink{structPosition}{Position}), unde primul segment este capul șarpelui.

\begin{DoxyNote}{Note}
Capul șarpelui este întotdeauna la indexul 0 în vectorul de segmente 
\end{DoxyNote}
